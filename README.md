# AgentTerm

A terminal emulator designed for Claude Code on Windows (WSL).

## Download

Get the latest release from the [Releases page](https://github.com/yunxin/agent-term-public/releases).

### Windows

- **Installer** (`AgentTerm-x.x.x-setup.exe`): Standard Windows installer
- **Portable** (`AgentTerm-x.x.x.exe`): No installation required, run directly

## Install

1. Download the latest `.exe` from the [Releases page](https://github.com/yunxin/agent-term-public/releases)
2. Run the installer or portable executable
3. AgentTerm launches directly into a WSL shell
